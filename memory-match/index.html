Memory Match Game

How to Play

When the game starts, several squares will briefly flash on the screen. Your task is to remember the position of these squares.

After a short delay, the squares will return to their normal state. Click the squares you remember.

Each round, the number of squares will increase, making it more challenging. If you select an incorrect square, the game will end. Match all the squares correctly to win!
